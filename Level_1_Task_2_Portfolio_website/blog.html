Full Stack Project - Social Media Application

Desc : This project aims to create a Social Media Application, inspired by popular platforms like Instagram. The goal is to provide users with an engaging and interactive experience.
Here's an overview of the approach I took:
1. Frontend Development: - Created the frontend using React.js, focusing on building an intuitive and user-friendly interface.
2. Backend Development: - Developed the backend using Node.js to handle server-side functionalities and data management.
3. Integration: - Integrated the frontend and backend, ensuring seamless communication and data flow between the two.
4. Authentication: - Implemented an authentication system to secure user accounts and ensure privacy.
Notable features of the project include:
- MERN Stack: The project utilizes the MERN (MongoDB, Express.js, React.js, Node.js) stack, allowing for efficient and scalable development. - User Interaction: Users can follow other users, create and like posts, and view posts from people they follow. - Logout Functionality: Users can log out of their accounts to ensure privacy and security. - Profile Customization: Users have the option to update their profile picture and personal details.
